# Console Banking Application (C#)

A simple **console-based banking system built using C#**.  
This project simulates basic ATM banking operations such as **login, deposit, withdrawal, and balance checking**.
**MY FIRST C# PROJECT!!**
The application demonstrates the use of **object-oriented programming, lists, user input handling, and basic banking logic**.

---

## Features

- Account Login using **Account Number and PIN**
- Deposit Money
- Withdraw Money
- Check Account Balance
- Simple console menu navigation
- Multiple bank accounts stored in a list
- Basic validation for withdrawals

---

## Technologies Used

- C#
- .NET Console Application
- Object-Oriented Programming (OOP)

## How the Application Works

1. The program starts and displays a welcome message.
2. The user enters their **Account Number**.
3. The user enters their **PIN**.
4. If authentication is successful, the user can:

- Deposit money
- Withdraw money
- Check their balance
- Exit the application

5. The program performs the selected banking operation and updates the account balance.
